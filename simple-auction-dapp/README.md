## Getting Started

1. Clone the Repo

2. Enter the project directory:
    ```bash
    cd simple-auction-dapp
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```

4. Run the development server:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
